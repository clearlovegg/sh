## 1.git基础命令
```
git init 初始化
git config user.name xxx 可以全局
git config user.email xxx
git add . 
git add [file_name]
git commit -m  "name"
git remote add origin https://clearlovegg@github/clearlovegg/shell.git
git pull origin master
git push origin master
git fetch origin master 取回
git merge origin/master 合并
git config --list   查看工作环境信息
git status 状态
git checkout -- [file] 恢复误删的文件
```
